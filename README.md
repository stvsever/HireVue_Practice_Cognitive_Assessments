# HireVue Practice Games

A small set of browser-based practice games. Open the main page and choose a game.

Navigation
- Use the back-arrow in the top-left to return to the main.

General
- Total game time: 3 minutes per game.
- Sounds can be toggled on/off.
- Correct answers advance difficulty; wrong answers retry the same level.

Digitspan
- Memorize a sequence of digits and enter them back using the on-screen keypad.
- Reveal modes: all-at-once, one-by-one, or halves (first half then second half).
- Difficulty increases by increasing the digit span over time.
- Optional reverse mode: enter the digits in reverse order.
- Answer window: ~10 seconds after the reveal completes. Auto-submits when all digits are entered.

Numerosity
- Operation target puzzle. Pick numbers that satisfy the displayed operation and target.
- Operations include addition (+), subtraction (−), multiplication (×), and integer division (÷).
- Some rounds require 2 numbers; later rounds may allow/require 3 numbers.
- Per-round timer: ~30 seconds.

Shapedance
- Select all tiles that match the hidden target arrangement; matches are rotation-invariant.
- Tiles may be randomly rotated or slowly spinning; focus on the pattern layout.
- Per-round timer: ~30 seconds.

Local Development
- Open `index.html` in a browser, or serve the directory with any static server.
- For GitHub Pages, push these files and enable Pages.
